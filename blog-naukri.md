## Projects Section

**Project Name:** open-mlpipe

**Description:**
Production-level automated ML pipeline — from raw data to deployed model in one line.

**Key Features:**
- 12-stage automated ML pipeline (Load → EDA → Clean → Feature Eng → Split → Preprocess → Compare → Tune → Select → Evaluate → Explain → Save)
- 14+ model types: Ridge, Lasso, ElasticNet, DecisionTree, RandomForest, ExtraTrees, XGBoost, LightGBM, GradientBoosting, HistGradientBoosting, AdaBoost, KNN, SVM, Stacking, Voting
- Auto EDA with statistical tests and VIF analysis
- Feature engineering: interactions, log transforms, missingness flags, datetime extraction
- Hyperparameter tuning with Optuna (18 model-specific search spaces)
- SHAP explainability (global + local feature importance)
- Full inference pipeline saved (feature engineering + model)
- Overfitting/underfitting detection
- Production metrics: R², RMSE, MAE, MAPE, F1, ROC-AUC, MCC
- 106 unit tests + 3 integration tests

**Technologies:** Python, scikit-learn, XGBoost, LightGBM, Optuna, SHAP, Pydantic, Click, Rich, pandas, numpy

**Links:**
- PyPI: https://pypi.org/project/open-mlpipe/
- GitHub: https://github.com/loisekk/open-mlpipe

**Role:** Creator & Maintainer
