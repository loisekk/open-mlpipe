    Production-level automated ML pipeline — from raw data to deployed model in one line.

    Quick Start¶

    pip install open-mlpipe

    from open_mlpipe import run

# One line — full pipeline
ctx = run("dataset.csv", target="price")

# Access results
print(f"Best model: {ctx.best_model_name}")
print(f"Test R²: {ctx.metrics['test_r2']:.4f}")

    What It Does¶

    Feature Description
    14+ Models Ridge, Lasso, XGBoost, LightGBM, RandomForest, and more
    Auto EDA Column hygiene, quality, distributions, statistical tests
    Feature Engineering Interactions, log transforms, missingness flags
    Hyperparameter Tuning Optuna with 18 model-specific search spaces
    SHAP Explainability Global + local feature importance
    Production Inference Full pipeline saved (feature engineering + model)

    Installation¶

    pip install open-mlpipe

    Optional extras:

    pip install open-mlpipe[catboost]    # CatBoost support
pip install open-mlpipe[mlflow]      # MLflow tracking
pip install open-mlpipe[deploy]      # FastAPI + Docker
pip install open-mlpipe[full]        # Everything
